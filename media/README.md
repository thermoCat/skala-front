# media

여행 앨범(사진+영상+음악 묶음)은 **두 가지 방법**으로 넣을 수 있습니다.

## 1. 브라우저에서 직접 올리기 (권장)

로그인한 뒤 `myTrip.html`에서 올립니다. 파일을 이 폴더에 넣을 필요가 없습니다.

1. `logIn.html`에서 로그인
2. `myTrip.html`로 이동
3. **"내 컴퓨터에서 사진 올리기"** 를 누르거나, 사진을 그 영역에 끌어다 놓기 (여러 장 가능)
4. 뜨는 창에서 대표 사진(썸네일)을 지정하고, 필요하면 영상·음악을 추가해 제목·내용과 함께 게시
5. 메인 화면의 썸네일을 누르면 사진·영상·음악을 한 앨범으로 모아 볼 수 있고,
   대표 사진 변경·제목/내용 수정·앨범 삭제도 그 창에서 할 수 있음

올린 앨범은 브라우저(localStorage)에 계정별로 저장되므로,
로그아웃했다 다시 로그인해도 그대로 남고 회원마다 앨범이 다릅니다.

### 알아둘 점

- 사진은 원본을 그대로 저장하면 공간이 금방 차기 때문에, 올릴 때 **긴 변 1600px / JPEG 품질 80%** 로
  자동으로 줄여서 저장합니다 (한 장에 200KB 안팎).
- 영상·음악은 압축 없이 그대로 저장되므로 **영상 4MB, 음악 3MB** 크기 제한을 둡니다.
- 브라우저 저장 공간은 보통 5~10MB라 앨범을 몇 개 이상 만들면 금방 찹니다.
  가득 차면 안내 메시지가 뜨고, 기존 앨범을 몇 개 지우면 다시 올릴 수 있습니다.
- 저장 위치가 브라우저라서 **다른 브라우저나 시크릿 모드에서는 보이지 않습니다.**
  파일이 아닌 브라우저 데이터이므로 git에도 올라가지 않습니다.

## 2. 모든 회원이 기본으로 갖는 SKALA 앨범

`js/auth.js`의 `Auth.getAlbums(userId)`는 회원이 앨범을 한 번도 손대지 않은 상태로
처음 조회되면 아래 파일들로 구성된 "SKALA 캠퍼스 여행" 앨범을 자동으로 만들어 저장한다
(새로 가입한 회원은 물론, 이 기능이 생기기 전부터 있던 회원도 처음 조회되는 시점에 똑같이
생긴다). 과제 스펙이 요구하는 `<img>`, `<audio><source>`, `<video><source>` 요소를
누구나 로그인만 하면 바로 확인할 수 있게 하는 용도다. 회원이 이 앨범을 지우면 다시
자동으로 채워 넣지 않는다(회원의 선택을 존중).

| 파일명 | 설명 | 상태 |
| --- | --- | --- |
| `skala-banner.jpg` | SKALA 소개 배너 ("AI 시대, 미래 인재로 성장하는 길") | ✅ 넣어둠 |
| `sk-logo.png` | SK 로고 | ✅ 넣어둠 |
| `skala-song.mp3` | 기본 배경 노래 (샘플, 9초) | ✅ 넣어둠 |
| `skala-vlog.mp4` | 기본 브이로그 영상 (샘플, 10초) | ✅ 넣어둠 |

`skala-song.mp3`, `skala-vlog.mp4`는 저작권 문제가 없는 [samplelib.com](https://samplelib.com/)의
"no license restrictions" 샘플 파일을 임시로 받아 넣어둔 것입니다. 나중에 실제 SKALA 테마곡·영상으로
교체해도 됩니다.

사진은 가로:세로 16:10 비율을 권장합니다 (다른 비율이어도 잘리지 않고 맞춰집니다).
영상·음악은 용량이 크지 않은 짧은 파일을 권장합니다.
이름이나 설명을 바꾸려면 `js/auth.js`의 `defaultAlbum()` 함수를 고치면 됩니다.
