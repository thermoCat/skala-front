# media

여행 사진은 **두 가지 방법**으로 넣을 수 있습니다.

## 1. 브라우저에서 직접 올리기 (권장)

로그인한 뒤 `myTrip.html`에서 올립니다. 파일을 이 폴더에 넣을 필요가 없습니다.

1. `logIn.html`에서 로그인
2. `myTrip.html`로 이동
3. **"내 컴퓨터에서 사진 올리기"** 를 누르거나, 사진을 그 영역에 끌어다 놓기
4. 여러 장을 한 번에 고를 수 있고, 올린 뒤 설명을 눌러 바꾸거나 삭제할 수 있음

올린 사진은 브라우저(localStorage)에 계정별로 저장되므로,
로그아웃했다 다시 로그인해도 그대로 남고 회원마다 앨범이 다릅니다.

### 알아둘 점

- 원본을 그대로 저장하면 공간이 금방 차기 때문에, 올릴 때 **긴 변 1600px / JPEG 품질 80%** 로
  자동으로 줄여서 저장합니다 (한 장에 200KB 안팎).
- 브라우저 저장 공간은 보통 5~10MB라 **20~30장 정도**가 한계입니다.
  가득 차면 안내 메시지가 뜨고, 몇 장 지우면 다시 올릴 수 있습니다.
- 저장 위치가 브라우저라서 **다른 브라우저나 시크릿 모드에서는 보이지 않습니다.**
  파일이 아닌 브라우저 데이터이므로 git에도 올라가지 않습니다.

## 2. 이 폴더에 예시 사진 두기

로그인하지 않은 방문자에게 보여줄 예시 사진입니다.
아래 이름 그대로 저장하면 `myTrip.html`과 `index.html`(여행 앨범 한 컷)에 나타납니다.

| 파일명 | 설명 |
| --- | --- |
| `lijiang-gorge.jpg` | 리장 호도협 |
| `cebu-night.jpg` | 세부 밤 스카이라인 |
| `hokkaido.jpg` | 홋카이도 대설산 |

가로:세로 16:10 비율을 권장합니다. 다른 비율이어도 잘리지 않고 맞춰집니다.
이름이나 설명을 바꾸려면 `html/myTrip.html`의 `SAMPLE_PHOTOS`와
`html/index.html`의 `SAMPLE_ALBUM`을 함께 고치면 됩니다.
